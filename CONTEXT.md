# IBD RS Rating

미국 주식 약 4,600개의 IBD 스타일 상대강도(Relative Strength) Rating을 매 거래일 계산해 Supabase(PostgreSQL)에 축적하는 시스템.

## Language

**RS Raw (상대강도 원점수)**:
한 종목의 가중 모멘텀 점수. 4개 기간 수익률(ROC)의 가중합 = 0.4×ROC(63거래일) + 0.2×ROC(126) + 0.2×ROC(189) + 0.2×ROC(252). 백분위로 변환되기 전의 원시 값. 각 ROC는 **그 종목의 실제 거래일** 기준 n거래일 전 종가 대비 수익률(달력 행 기준이 아님).
_Avoid_: momentum score, raw score

**RS Rating (상대강도 등급)**:
RS Raw를 같은 거래일의 **모집단** 전체에 대해 백분위 순위로 환산한 1~99 정수. 99 = 상위, 1 = 하위.
_Avoid_: percentile, rank, score

**모집단 (Universe)**:
특정 거래일에 RS Rating을 매길 때 비교 대상이 되는 종목 집합. 원칙: 그날 충분한 히스토리(252거래일)를 가진 모든 종목(약 4,600개) + 기준 지수. 유효 종목이 임계치(예: 90%) 미만인 날은 RS Rating을 산출하지 않는다.
_Avoid_: pool, denominator, 분모 (한글은 "모집단"으로 통일)

**Reference ticker (기준 지수)**:
SPY, QQQ. 일반 ETF는 유니버스에서 제외하지만 이 둘은 지수 대표로 **모집단에 포함**해 RS Rating까지 부여한다 — "지수가 전체 종목 중 상위 몇 %인지" 확인하는 용도. (4,600개 중 2개라 다른 종목 백분위 왜곡은 무시할 수준.)
_Avoid_: benchmark only (순위에서 빼는 의미로 오해 금지)

**워밍업 (Warm-up)**:
RS Raw 산출에 필요한 최소 히스토리(252거래일 ≈ 1년)가 아직 쌓이지 않은 구간. 이 구간의 종목은 RS가 없다. 실데이터상 RS Rating은 2025-03-21부터 등장.

**정체 (Starvation)**:
글로벌 최신-날짜 커서가 일부 종목만 전진시켜, 뒤처진 종목이 영구히 갱신되지 않는 버그 상태. 2026-04-17 이후 약 97% 종목이 이 상태였다.

**조용한 정체 (Silent starvation)**:
워크플로우는 "성공(초록)"으로 끝나는데 실제로는 일부 종목만 갱신되는 상태. 실패 이메일이 오지 않아 가장 위험하다. 데이터 완전성 자가 점검으로 탐지해 워크플로우를 실패 처리한다.

**트레일링 윈도우 (Trailing Window)**:
정체를 막기 위해 매 거래일 모든 종목에 대해 "최근 N거래일(7~10일)"을 일괄 재요청·덮어쓰는 다운로드 방식. 부분 실패가 다음 실행에서 자동 치유(self-healing)된다. 글로벌 최신-날짜 커서를 대체한다.

## Relationships

- 한 **종목(ticker)**은 거래일마다 하나의 **RS Raw**와 (모집단이 충분하면) 하나의 **RS Rating**을 가진다.
- **RS Rating**은 같은 거래일 **모집단** 전체의 **RS Raw** 분포에 대한 백분위다 — 즉 한 종목의 등급은 그날 다른 모든 종목에 의존한다.
- **종목 유니버스**는 Finviz 스크리너가 반환하는 ~4,600 종목 + **Reference ticker**로 구성된다.

## Flagged ambiguities

- **모집단** 정의: "그날 종가가 있는 종목" vs "충분한 히스토리를 가진 전체 유니버스" → 후자로 해결(2026-06-07). 불완전한 날은 RS 미산출.
- **Reference ticker** 순위 처리: config 주석("제외")과 코드("포함")가 모순 → "포함(함께 랭크)"으로 해결(2026-06-07). 코드 유지, 주석 수정.
- **실패 알림**: 진단 시 "알림 없음"으로 오판 → GitHub 기본 이메일 알림이 이미 작동함을 확인. 빈틈은 "조용한 정체"(성공으로 표시되는 정체)이며, 자가 점검으로 워크플로우를 실패시켜 기존 이메일이 잡게 한다(2026-06-07).
